LES DEFINITIONS

FIGMA

Figma est un site libre de droit et d'utilisation pour la création de maquette d'un futur site internet. Cela permet de visualiser le projet web afin de coder plus facilement.

MAMP

Mamp est le néologisme basé sur LAMP (Mac Apache Mysql Php)

WORD PRESS

Word press est un CMS (content managment system) ce qui permet de recuperer des temlates complet de site internet

FILEZILLA

Filezilla est un logiciel de partage de donné d'ordinateur à ordinateur

GIT

git est Un logiciel de gestion de versions décentralisé il permet de créer et de gérer les différentes version de notre projet en permettant a tout moment de revenir sur des versions antérieure du projet, il facilite également le travail de groupe.

VISUAL STUDIO CODE

Visual studio code se traduit par l'anagrame IDE (integrated development environment) C'est un outil environnemental digital pour le developpement de code

NODE

Node.js est un environnement bas niveau permettant l'exécution de JavaScript côté serveur.
