LES DEFINITIONS

FIGMA

Figma est un site libre de droit et d'utilisation pour la création de maquette d'un futur site internet. Cela permet de visualiser le projet web afin de coder plus facilement.

MAMP

Mamp est le néologisme basé sur LAMP (Mac Apache Mysql Php)

WORD PRESS

Word press est un CMS (content managment system) ce qui permet de recuperer des temlates complet de site internet
